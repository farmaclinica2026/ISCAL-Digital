# ISCAL Digital

Ferramenta de consulta rápida para profissionais de saúde da **Irmandade da Santa Casa de Londrina (ISCAL)**.

## Funcionalidades

- **Profilaxia Antibiótica Perioperatória**: Protocolos baseados na Norma Técnica SPCIH-015 (Versão 06 - Rev: 03/09/2025).
- **Ajuste Renal de Antimicrobianos**: Calculadora de função renal (CKD-EPI 2021) com ajuste por BSA e sugestões de posologia para diversos antimicrobianos, antifúngicos e antivirais.
- **Navegação Integrada**: Suporte ao botão "Voltar" do navegador para uma experiência fluida.

## Tecnologias Utilizadas

- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (Ícones)
- **Vite** (Build tool)

## Como Executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para gerar a versão de produção:
   ```bash
   npm run build
   ```

## Aviso Legal

Este aplicativo é uma ferramenta de consulta rápida para profissionais de saúde, baseado em diretrizes institucionais e bases de dados de referência. O uso é exclusivo para profissionais e a decisão clínica final é sempre de responsabilidade do médico assistente.
